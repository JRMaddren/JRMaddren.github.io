Explore Camp

Key Features

This was an excellent frist project that I completed while doing my Web Development Bootcamp with Udemy.

Full CRUD on the show page, Authentication and Authorization with the login and logout function. I have also implemented a Mapbox Cluster Map that will show the campsites with locations. Users can review and upload their own campsites with images and also have the option to delete their own campsite or uploaded images.

A short rundown of all the technologies I used in this project are as follows. HTML, CSS & Bootstrap for the markup and front-end. I used Responsive Design, Media Queries and FlexBox for the cards to display the individual campsites and their information. The backend was written in NodeJS with Express and MongoDB for my NoSQL Database.
